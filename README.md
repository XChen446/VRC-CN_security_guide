# VRC-CN_security_guide
因社区局势愈发紧张而产生的VRC中文社区安全指南仓库
## :information_source: 社区安全指南
近期VRchat社区内活跃着一批 ***不明生物*** ，对正常游戏环境的破坏极大，

根据相关信息提供者披露的内容，现公布部分早期信息 *十二上旬左右*

[Discord](https://discord.com/ "对应频道邀请链接已失效")上出现了一个分发扫模、热上传插件的频道，并在视频中教授其他玩家如何操作这些软件，并每晚固定直播，

内容为：

- 当面扫模
- 当面骚扰
- 给其他萌新安利盗模工具
- 给他们上传盗来的模型
- 高强度巡逻b站、贴吧、qq群，举报和攻击相关揭露其行为的个人或社交账号
- 对其言论特别不满意的严重的会在直播中将其开盒，挂在dc频道上
  
  *（因需要控制事态扩大化，相关记录已经被删除，但是已保留相关证据，各位极大概率可以在自己的社交圈子内拿到关键证明，文档结尾仅展示部分）*

通过著名开膛手Nx提供的付费客户端，即可绕过EAC为vrchat客户端添加各种作弊插件，当然也包括获取未经本地加密的vrca文件，此时直接通过ARC即可跳过编译步骤直接上传模型，这就是所谓的当面强行复制私人模的由来。

### :warning: 该活动已经常态化，近日直播常见做法为：
> 1. “你的模型好好看，能不能给我复制” *此时已在后台把模型扫下来如果拒绝，热上传完毕后然后当面切出来给你看*
> 2. “你听说最近有人在盗模，你信吗？那是假的，是贩子编造的谎言”
> 3. “哎呀是有不怀好意的人在刻意营造焦虑，我是好人”
> 4. “好好笑，让大伙都不去公开房真牛逼，那还玩什么游戏”

## 不要在VRC bio中挂任何自己的社交信息 :warning: :warning: :warning:
非常不建议前往公开房（或者进去前切换成公开模），

### 警惕身边有相关不认识或莫名上来搭话的人，并根据 **[劣迹玩家榜单](./malicious_users.md)及时拉黑相关人员**

警惕qq群有相关言论或为其洗白的的人，该团伙在国内伙同其招揽进社区的玩家，

高强度巡逻，举报揭露其行为的相关帖子和视频

### 关于大规模封号问题

封号本文档从未提及过，但历史上vrchat确实现过大规模无差别无理由封号，有服务器判定规则改变、小团体疯狂举报等。

本次相关案例疑似为：
- vrchat为vket的到来大规模严厉打击nsfw而被连带；
- 新更新的用户条款开放了一个管理员api，使得vrchat官方可以不经过任何玩家个人许可直接访问他们认为的“侵害行为”事件发生的位置并对相关玩家进行直接处理，该端口疑似遭到泄露。（2025.12.20）
